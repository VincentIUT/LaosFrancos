--------------------- PROJET SITE WEB ---------------------

Site pour une association Franco-Laotienne

Le but est de proposer une plateforme d'échange
entre français et laotiens, laquelle est basé sur
l'existence d'un café culturel permettant l'organisation
d'évènements et de cours de laotien.
Le site offre un forum de discution ainsi qu'un galerie
pour poster ses meilleures photos avec la communauté.
